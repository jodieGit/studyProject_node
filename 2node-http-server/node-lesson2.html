Window下创建第一个应用

  • 1.创建server.js文件
  • 2.cmd -> node setver.js(文件放在指定目录下面，如果不在，会提示相应的目录)
  • 3.在浏览器中输入server.js里面的指定url

Mac下创建第一个应用

  • 1.Command + 空格，输入终端
  • 2.vim server.js,输入server.js的相关内容，注意ip地址进行修改（i 切为插入模式）；EXC退出插入模式，:wq保存修改
  • 3.node server.js
  • 4.在浏览器中输入server.js里面指定的url

ls:查看当前目录，rm server.js 删除server.js文件
